Creating a Node Project on Heroku
  • Why do it?
    Heroku is a popular platform as a service (PaaS) offering that allows web developers to host applications in expandable containers. You can add services such as databases and worker processes to your application giving you a complete solution without having to deal with setting up your own servers.
  • What should I already know?
    In order to be successful, you should:
    • Understand how to use the command line on your computer
    • The basics of working with git (optional)
    • The basics of working with the Node Package Manager (NPM)
  • What do I need?
    To work with Heroku, you need:
    • A working Internet connection on your computer
    • The Heroku command line client installed on your computer
    • A git command line client installed on your computer
    • Node / npm installed on your computer
